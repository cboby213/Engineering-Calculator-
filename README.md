# 🧮 Engineering Calculator

A modern **Engineering (Scientific) Calculator** built using **HTML, CSS, and JavaScript**.
It supports advanced mathematical operations and provides a clean, responsive user interface.

---

## 🚀 Features

* Basic arithmetic operations (+, −, ×, ÷)
* Scientific functions:

  * sin, cos, tan
  * log (base 10)
  * square root (√)
  * power (xʸ)
  * π (Pi value)
* Clean and modern UI with gradient background
* Fully responsive design
* Single file implementation (easy to use)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
